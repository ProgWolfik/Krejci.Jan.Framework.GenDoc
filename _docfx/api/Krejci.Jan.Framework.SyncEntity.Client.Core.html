Namespace Krejci.Jan.Framework.SyncEntity.Client.Core

Classes

SyncManager

Manages thread-safe access to synchronization stores keyed by entity type. Supports applying entity changes and initializing stores with existing entities. Ensures thread safety when accessing internal collections, while assuming individual store operations are themselves thread-safe.

SyncStore<T>

A thread-safe generic store for synchronizing entities of type T. It maintains an internal dictionary keyed by entity ID and supports applying synchronization actions such as add, update, and delete based on serialized JSON data.

Interfaces

ISyncStore

Interface defining a synchronization store capable of applying sync actions and initializing with a collection of entities.
