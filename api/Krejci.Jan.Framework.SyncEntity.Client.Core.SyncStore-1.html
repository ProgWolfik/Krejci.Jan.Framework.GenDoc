Class SyncStore<T>

Krejci.Jan.Framework.SyncEntity.Client.Core

Krejci.Jan.Framework.SyncEntity.Client.Core.dll

A thread-safe generic store for synchronizing entities of type T. It maintains an internal dictionary keyed by entity ID and supports applying synchronization actions such as add, update, and delete based on serialized JSON data.
