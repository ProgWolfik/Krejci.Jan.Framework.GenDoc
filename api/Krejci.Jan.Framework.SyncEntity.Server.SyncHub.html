Class SyncHub

Namespace
Krejci.Jan.Framework.SyncEntity.Server
Assembly
Krejci.Jan.Framework.SyncEntity.Server.dll

SignalR hub responsible for managing client connections and grouping them by user roles. When a client connects, they are automatically added to a group corresponding to their role. When a client disconnects, they are removed from that group. This allows broadcasting messages selectively to users based on their roles.
